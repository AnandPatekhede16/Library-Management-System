Welcome to the Library Chat! Connect via a TCP client to port 9090 to start chatting.
How to connect:
Windows: telnet localhost 9090
Mac/Linux: nc localhost 9090
First line you send = your username.
About the Chat System

This chat uses a multi-threaded Java Socket Server running on port 9090. Each connected client gets its own thread from a fixed thread pool. Messages are broadcast to all connected users in real-time.

Connect via Terminal
# Windows (enable telnet in features first)
telnet localhost 9090
# Mac / Linux
nc localhost 9090
# Then type your username and press Enter
YourName
# Now type messages freely!
